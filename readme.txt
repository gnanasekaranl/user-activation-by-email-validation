Require users to click the activation link to access the site which is emailed upon user registration.

== Description ==

Activation mail sent to the new user email once a user registered. The user must activate the account by click activation link in addition to a username and password to log in successfully the first time. 

This plugin does not currently work with multi-site. Multi-site uses an activation key system for new registrations already.

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
